# challenge

This is a simple RESTful service for managing accounts and handling money transfers between accounts. The service is built using Spring Boot and includes basic functionality for creating accounts, transferring money, and sending notifications. The project is designed to be thread-safe and to prevent overdrafts during transfers.


## Getting Started

### Prerequisites

- Java 11
- Gradle 7.x
- An IDE that supports Lombok (e.g., IntelliJ IDEA or Eclipse)

### Building and Running the Project

1. **Clone the repository:**
2. **./gradlew build
3. **./gradlew bootRun
